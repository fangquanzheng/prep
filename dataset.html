Customers Table

Suppose we want to get the data from Orders and Customers with same CustomerID
The inner JOIN will be the proper way to do this

SELECT Orders.OrderID, Customers.CustomerName
FROM Orders
INNER JOIN Customers
ON Orders.CustomerID = Customers.CustomerID;

And there are the result below

The inner JOIN will only select the data from Orders Table and Customers Table which they has the same CustomerID
In this case, only the data with CustomerID 2.
And only the OrderID and CustomerName of this data will be shown from this particular function.
