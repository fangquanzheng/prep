How SQL INNER JOIN works

The INNER JOIN keyword selects records that have matching values in both tables. The syntax for joining two tables are as followed.

SELECT column_name(s)
FROM table1
INNER JOIN table2
ON table1.column_name = table2.column_name;

Which basically combine the different information in the two tables and combine them together. And INNER JOIN only selects all rows from both participating table when there is a match between the columns.
